**接口地址**: `POST /v1/common/address/legal`

检验地址格式合法性，可在发起提币前对接收方地址进行校验。（此接口不需签名）

## 请求参数

| 参数       | 数据类型 | 是否必须 | 描述         |
|------------|----------|----------|--------------|
| chainName  | String   | True     | 链           |
| address    | String   | True     | 区块链地址   |

## 响应参数

`data` 字段的响应参数：

| 参数   | 数据类型 | 是否必须 | 描述                      |
|--------|----------|----------|---------------------------|
| result | Bool     | True     | False 为不合法,True 为合法 |

## 响应示例

```json
{
    "code": 200,
    "msg": "success",
    "timestamp": "1725619914435",
    "data": {
        "result": true
    }
```
