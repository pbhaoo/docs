### 接口地址  
**GET** `/v1/account/withdraw/query`

### 请求参数

| 参数        | 数据类型 | 是否必须 | 描述                                     |
|-------------|----------|----------|--------------------------------------------|
| id          | String   | False    | 平台返回的订单id                          |
| businessId  | String   | False    | 商户自定义的业务id                        |
| timestamp   | Long     | True     | unix时间戳(毫秒)                          |

注意：`id`或`businessId`只能传其中一个。

### 响应参数

| 参数          | 数据类型 | 是否必须 | 描述                      |
|---------------|----------|----------|---------------------------|
| id            | String   | True     | 订单ID                    |
| tokenName     | String   | True     | 币名                    |
| chainName     | String   | True     | 链名                      |
| toAddress     | String   | True     | 接收方地址                |
| amount        | Decimal  | True     | 提币数量                  |
| status        | String   | True     | 交易状态，具体见“枚举定义”中的解释|
| businessId    | String   | False    | 商户自定义的业务ID，如有   |
| tag           | String   | False    | 订单备注             |
| txId          | String   | False    | 链上交易哈希                  |
| blockHigh     | String   | False    | 区块高度                  |
| networkFee    | String   | True     | 收取的网络手续费                |
| platformFee   | String   | True     | 收取的平台佣金               |
| createTime    | String   | True     | 创建时间(UTC)             |
| completeTime  | String   | False    | 完成时间(UTC)             |
