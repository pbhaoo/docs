### 请求格式

所有的 API 请求都是 RESTful,目前仅支持两种方法:GET 和 POST。

*  **GET 请求**：所有的参数必须都在路径参数里。
* **POST 请求**：所有参数通过 request body 以 payload 发送。

### 接入 URL

**https://api-beta.baaswallet.io/baaswallet/api/**


### 请求鉴权

- 所有的接口都需要 API-Key,在请求头中以 `X-BS-APIKEY` 字段传递；
- 每个 API-Key 可以对接口权限进行配置，默认所有接口不能访问。请求前请先确保 API-Key 先有对应的访问权限。

### 请求限频

对某单个节点请求的频率不能超出单位时间内该节点最大允许访问次数的限制。
违反任何一个速率限制的时候，将返回：429
