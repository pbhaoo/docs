如需使用 API,请先登录网页后台,完成 API Key 的申请和权限配置，再依据文档详情进行开发和交易。您可以点击 [这里](https://console-beta.basswallet.io/#/apiManagement/index) 创建 API Key。

一个子帐号最多可申请 10 个 API Key,创建成功后务必记住以下信息：
  - **Access Key**(API 访问密钥)
  - **Secret Key**(签名认证加密所使用的密钥，仅申请时可见，如丢失，需重新申请)

**安全提示**
  - 永远不要把您的 API Key/Secret 告诉任何人。如果不小心泄露，请立即删除此 Key。
  - 每个 API Key 最多可绑定 20 个 IP 地址（主机地址或网络地址）。出于安全考虑，强烈建议您绑定 IP 地址。未绑定 IP 地址的 API Key 无法开通提现权限。
