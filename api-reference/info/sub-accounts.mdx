---
title: '子账户'
description: ''
---

- 当你有多个业务需要独立进行资产管理时，使用子账户是一个很好的方法。各个账户之间的资金和API Key都是相互独立/隔离的；
- 资产可以在主账户和子账户之间进行划转，子账户之间无法进行划转；
- 子账户拥有独立的账号/密码 和 API Key，均由主账户在商户后台进行配置管理；
- 每个主账户可创建20个子账户，每个账户最多可创建20组Api Key，每个Api Key可对应配置读取、提现等权限。

您可以点击[这里](https://console-beta.basswallet.io/#/subAccount/index) 进入到创建子账户的页面。
